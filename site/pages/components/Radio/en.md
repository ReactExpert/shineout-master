## API

Radio cannot be used alone.

### Radio

| Property | Type | Default | Description |
| --- | --- | --- | --- |
| checked | bool \| 'indeterminate' | - | if not set, use (value === htmlValue). |
| disabled | bool | false | disable checkbox |
| htmlValue | any | true | Specifies the result |
